## 2. 执行签到
```shell
curl http://81.68.144.211:5002/yaduoapi/daily_signin?token=&lot_number=&pass_token=&gen_time=&proxies=&captcha_output=
```
URL参数说明
- `token`：亚朵账号token
- `lot_number`、`pass_token`、`gen_time`、`captcha_output`: 为环节1中获取到的验证码参数
- `proxies`：为http代理

返回示例：

- 签到成功
```json
{
  "module": "daily_signin",
  "function": "export_signin",
  "data": {
    "result": {
      "debrisDesc": "获得1消费积分、1张【竹居】拼图碎片",
      "needLottery": false
    },
    "retcode": 0,
    "retmsg": "操作成功"
  }
}
```

- 重复签到
```json
{
  "module": "daily_signin",
  "function": "export_signin",
  "data": {"result":null,"retcode":1000,"retmsg":"今天已经签到过了"}
}
```
