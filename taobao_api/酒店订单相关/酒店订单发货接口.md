taobao.xhotel.order.update( 酒店订单发货接口 )

## 请求参数

| 名称                           | 类型   | 必须  | 示例值             | 描述                                                         |
| ------------------------------ | ------ | ----- | ------------------ | ------------------------------------------------------------ |
| tid                            | Number | true  | 100100100          | 订单号                                                       |
| opt\_type                      | Number | true  | 1                  | 操作的类型：1.确认无房（取消预订，710发送短信提醒买家申请退款）2.确认预订 3.入住 4.离店 5.noshow 6.关单 |
| sync\_to\_hotel                | String | false | Y                  | 是否把代理直签的订单同步到酒店，Y为同步，N不同步             |

| reason\_type                   | Number | false | 6                  | 取消类型，6 代表的是用户取消，reasonType=7代表的是小二协商   |
